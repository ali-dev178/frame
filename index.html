Exporting
How the quality is preserved: the original image is drawn into the canvas at a 1:1 pixel scale (no interpolation) and saved as PNG, which is lossless. For a standard photo without transparency, the picture area comes out pixel-identical to the file you put in. If a photo already matches the selected shape, the tool detects it, adds nothing, and downloading hands you the original file byte-for-byte — same format, same metadata, completely untouched. Platforms still apply their own compression when you upload — that part is outside any tool's control, but the file you hand them here is untouched.

About the video export: where the browser supports it (recent Chrome and Edge), export merges your frames and music directly into the MP4 with WebCodecs — no real-time recording, so a 60-second video takes only a few seconds. The MP4 muxing library ships inside the app, so export works fully offline. On browsers without WebCodecs the tool honestly falls back to real-time recording and says so. A single Still clip is copied 1:1 with zero resampling (odd dimensions get one duplicated edge row/column, never a removed one); captions and color looks draw over the frame by definition, and frames inside a transition window are blends of two clips — so choose Cut, Look: None, and no caption when strict frame-copies matter. Mixing clips of different sizes in one video scales the smaller ones up to the largest — for absolute crispness, use same-size clips. Video is a compressed format by nature, so it can't be bit-for-bit lossless like the PNG, but at Ultra (40 Mbps H.264, up to 320 kbps AAC — capped at what your system's encoder supports) a still image is far beyond any visible difference; your music is re-encoded to AAC, transparent at these bitrates.
